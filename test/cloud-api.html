OpenALPR Cloud API

OpenALPR Cloud API

The OpenALPR Cloud API is a web service running in the cloud that analyizes images of vehicles and responds with license plate data, as well as vehicle color, make, model and body type.

Take a Test Drive

Upload your own images, or select an image below to see the results from previous image analysis using the OpenALPR Cloud API.

Supported Countries

OpenALPR contains specialized training data for many plate styles. Currently, OpenALPR supports:

  • North America
  • Europe
  • Brazil
  • Australia
  • UAE
  • Saudi Arabia
  • India
  • Thailand
  • Argentina
  • New Zealand

If your country is not listed, you may still experience very high accuracy rates by using training data for a country that has plates that look similar to yours. Upload your own images in the form above to see how well OpenALPR will work for you. OpenALPR can be trained to accurately recognize license plates for any country. Contact Us to discuss custom training services.

Easy Integration

Because the OpenALPR Cloud API is REST-based, it works with virtually any programming language on any operating system. You can make API calls using whatever method you prefer.

To make integration easier, the OpenALPR Cloud API includes permissively licensed open source client libraries in a variety of languages. Head over to the OpenALPR Cloud API GitHub repo to find out more.

Check out the OpenALPR Cloud API Documentation for more technical information.
